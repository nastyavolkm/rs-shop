Что можно делать в приложении?
  • Выбрать своё месторасположение
  • Просмотреть каталог товаров по категориям, подкатегориям
  • Найти товар по ключевому слову
  • Без регистрации добавить/удалить товар в избранное, корзину
  • Войти в аккаунт/зарегистрироваться и получить возможность оформить заказ
  • Редактировать данные заказа и удалять заказ
